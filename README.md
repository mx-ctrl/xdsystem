---

📖 About

ENZOX is a lightweight, efficient command-line tool designed specifically for Termux users to monitor and analyze VPN network traffic.

---

⚙️ Requirements

Before proceeding, ensure your Termux environment meets these requirements:

Requirement Description
📱 Termux Installed on your Android device
🌐 Internet Active connection required
📦 Git Installed within Termux

---

🚀 Installation & Quick Start

Run this single command in Termux to download, configure, and launch ENZOX:

```bash
git clone https://github.com/mx-ctrl/xdsystem.git && cd xdsystem && chmod +x enzox && ./enzox
```

---

💻 Usage (Run Again)

If you've already installed ENZOX and want to run it again:

```bash
cd ~/xdsystem
./enzox
```

---

🛠️ Commands Summary

Command Description
git clone https://github.com/mx-ctrl/xdsystem.git 📥 Downloads the tool repository
cd xdsystem 📂 Navigates into the tool folder
chmod +x enzox 🔑 Grants executable permissions
./enzox ▶️ Runs the tool

🔧 Troubleshooting

1. Error: git: command not found
Install Git first:

```bash
pkg update && pkg install git -y
```

2. Error: Permission denied or No such file or directory
Verify you're in the correct directory and check files:

```bash
ls -la
```

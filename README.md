# ENZOX 🚀

ENZOX is a tool that runs directly on Termux.

## Install & Run

**One-liner copy-paste:**

```bash
git clone https://github.com/mx-ctrl/xdsystem.git && cd xdsystem && chmod +x enzox && ./enzox

Or use the installer:

bash install.sh

Run Again Later

cd ~/xdsystem && ./enzox

Requirements

Termux (latest version recommended)

Internet connection


> Git will be installed automatically if not present.



Folder Structure

~/xdsystem/
├── enzox       # Main binary
├── install.sh  # Installer script
└── README.md   # Documentation

Troubleshooting

Git not found:


pkg install git

enzox file missing:


cd ~/xdsystem && ls -la
